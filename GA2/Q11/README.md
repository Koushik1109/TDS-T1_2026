# GA2 Codespace devcontainer

Devcontainer configuration for GA2 (ga2-f478f0) with Python, uv, and FastAPI.

## Quick start

1. Push this repository to GitHub (create a new public repo)
2. Open the repo on GitHub → **Code** → **Codespaces** → **Create codespace on main**
3. Wait for the container to build (postCreateCommand installs FastAPI via uv)
4. In the Codespace terminal, run: `echo $GITHUB_REPOSITORY $GITHUB_TOKEN`
